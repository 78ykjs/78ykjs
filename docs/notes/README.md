# Notes

这里保存已经整理过的知识卡片。每篇笔记尽量围绕一个具体问题、概念、方法或经验展开。

## 分类建议

- `programming`：编程语言、框架、工程实践
- `data`：数据处理、实验、分析方法
- `research`：论文阅读、研究方法、写作
- `tools`：工具、脚本、自动化
- `thinking`：方法论、复盘、决策记录

## 笔记质量标准

- 标题能直接说明问题。
- 开头先给结论，后面再写过程。
- 至少补充一个“可复用经验”。
- 重要资料写清楚来源。
- 能关联项目或领域时补上链接。
