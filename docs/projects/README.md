## 项目记录结构

| 模块 | 记录重点 |
| --- | --- |
| 背景 | 为什么做，解决谁的问题 |

## 记录建议

- 项目开始时创建项目页。
- 关键问题和解决过程及时记录。
- 项目结束后补充复盘：做对了什么、踩了什么坑、下次如何改进。
