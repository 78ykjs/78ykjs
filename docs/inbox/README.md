# Inbox

这里放临时记录、灵感、待阅读材料和还没有整理好的内容。

## 使用规则

- 不要求格式完美，先记录下来。
- 每周至少清理一次。
- 有价值的内容移动到 `notes`、`projects`、`areas` 或 `resources`。

## 捕捉格式

```markdown
## 标题

- 来源：
- 时间：
- 关键词：

### 原始记录

### 为什么值得保留

### 下一步
```

## 分拣标准

| 去向 | 判断标准 |
| --- | --- |
| Notes | 能解释一个概念、问题或方法 |
| Projects | 和具体项目推进有关 |
| Areas | 属于长期能力建设 |
| Resources | 本身是外部资料、链接或工具 |
| 删除 | 一周后仍然没有价值或上下文 |
