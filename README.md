# 78ykjs

一个简洁主页 + 功能型知识库。

- 主页文件：[index.html](index.html)
- 知识库文件：[knowledge.html](knowledge.html)
- Pages 地址：`https://78ykjs.github.io/78ykjs/`
- 风格目标：主页简洁，一键进入知识库；知识库承载搜索、筛选、收藏、新增和导出。

## 知识库子域

- GitHub 搭建
- 毕业论文
- 风电故障诊断
- 数据实验
- 文献检索
- 自动化脚本

## 部署

推送到 `main` 后由 GitHub Actions 发布到 GitHub Pages。
