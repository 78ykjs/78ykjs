# 78ykjs

这里是我的 GitHub 个人主页，也是我的公开个人知识库入口。

我会把学习笔记、项目复盘、工具方法、论文资料、工程经验和长期主题整理在这里。这个仓库不追求一次性写完，而是作为一个可以持续生长、持续重构的数字花园。

## 博客入口

- [静态博客首页](index.html)
- GitHub Pages 启用后可通过仓库 Pages 地址访问。

## 知识库导航

- [知识库首页](docs/README.md)
- [收件箱](docs/inbox/README.md)：临时想法、待整理资料、碎片记录
- [笔记](docs/notes/README.md)：已经整理过的知识卡片
- [项目](docs/projects/README.md)：项目记录、开发过程、复盘总结
- [领域](docs/areas/README.md)：长期关注的方向与能力地图
- [资源](docs/resources/README.md)：书籍、论文、网站、工具和参考资料
- [模板](docs/templates/README.md)：笔记、项目、复盘和文献阅读模板

## 当前关注

- 软件工程与项目实践
- 自动化、脚本与效率工具
- 数据处理、实验记录与可复现工作流
- 论文阅读、知识整理与长期写作

## 维护方式

- 新想法先放入 `docs/inbox/`
- 能独立成文的内容整理到 `docs/notes/`
- 和具体成果相关的内容放入 `docs/projects/`
- 长期维护的主题沉淀到 `docs/areas/`
- 外部资料、工具链接和参考文献放入 `docs/resources/`
